# Text File Management System

A simple text file manager built using Node.js, Express, and EJS.

## Features
- Create text files
- Read files
- Edit (rename) files
- Delete files
- Error & success alerts (no extra pages)

## Tech Stack
- Node.js
- Express.js
- EJS
- Tailwind CSS
- File System (fs)

## How to Run
npm install
node app.js
